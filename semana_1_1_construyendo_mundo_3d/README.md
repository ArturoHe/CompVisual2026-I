## Descripción:

Con el desarrollo del taller se buscaba ampliar el entendimiento del estudiante sobre las estructuras básicas que componen los objetos 3d, para esto se desarrolló un modelo propio en blender, el cual fue exportado a unity para su visualización dentro del software, también se desarrollo un script con apoyo de la IA para hacer el conteo de las estructuras básicas.

## Implementaciones:

Para llevar a cabo el taller, se desarrolló un modelo tridimensional en Blender, utilizando las herramientas propias del software. Una vez completado, el modelo fue exportado en formato .obj para su importación en Unity. Con apoyo de ChatGPT, se desarrolló un script en Unity que permite analizar y visualizar la cantidad de vértices, triángulos y otros atributos del modelo, de esta forma logrando lo solicitado en la guía del taller.

## Aprendizajes y dificultades

Este primer taller me brindó un aprendizaje significativo respecto a Unity, aunque estoy un poco contextualizado ya que he usado software como Blender, Unreal Engine y un poco de SketchUp, nunca había usado Unity; por tal motivo se me dificultó un poco el desarrollo del taller, sin embargo, he logrado comenzar a aprender las cosas básicas del software. Debo comentar que, aunque ya sabía un poco de las estructuras básicas que componen un modelo 3d, al desarrollar este taller he aprendido nuevas cosas, en especial como Unity maneja los modelos (Yo desarrollé el modelo que usé para el taller).
